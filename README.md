# EduSphere ERP – School & College Management System

## Overview

EduSphere ERP is a web-based academic management system designed to simplify administration tasks such as student management, teacher records, attendance tracking, and fee management.

## Features

* Student Management
* Teacher Management
* Attendance Tracking
* Fee Management
* Reports & Analytics
* Admin Dashboard

## Technology Stack

* HTML5
* Bootstrap 5
* AdminLTE
* JavaScript
* ApexCharts

## Installation

1. Download or clone the repository

```
git clone https://github.com/username/edusphere-erp
```

2. Open the project folder

3. Run the system

```
open index.html
```

No additional configuration is required.

## Project Structure

```
/assets
/css
/js
/pages
index.html
```

## Deployment

The system can be deployed on:

* GitHub Pages
* Netlify
* Vercel
* Apache / XAMPP server
